# 免費的檔案存放平台
* 使用Google Workspace製作
* 每個次最多支援上傳10GB，不限上傳次數。
* 用方式分享檔案可以不浪費自己的Google儲存空間（以下會有教學）
* 上傳後會將內容回條寄送到您填寫的電子郵件地址。裡面會包含同個平台，其他使用者所上傳的內容。
* 每個表單存放的內容有一定大小上限，如果滿了無法上傳的話，請換一個表單試試。

* 您上傳的內容會在您的雲端硬碟儲存副本，若您決定刪除也不會影響此平台的內容。
